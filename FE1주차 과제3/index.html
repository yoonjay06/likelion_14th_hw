HTML

  • html 요소는 시작 태그와 종료 태그, 태그 사이에 위치한 내용이 존재
  • 앵커 태그 : a태그는 특정 위치로 이동, href속성은 하이퍼링크 경로를 입력
  • 목록태그 : 순서x=ul 순서o=ol

css
  • 인라인방식:HTML요소(태그)의 style속성에 직접 작성하는 방식
  • 내장(embedding)방식:HTML style안에 작성하는 방식
  • 링크방식:HTML link를 이용하여 외부 문서로 css를 불러와 적용하는 방식
  • @import방식: css@import를 이용하여 외부문서로 css를 불러와 적용하는 방식
